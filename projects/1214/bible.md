# 书名：《全民觉醒：我的傀儡全是神级BOSS》

## 核心卖点
1.  **极度反差与逆袭**：主角觉醒了公认的烧钱废物职业“傀儡师”，却觉醒了唯一神级天赋“万物提炼”，将原本需要昂贵材料制作的傀儡，变成了可以直接提取被杀死的魔物BOSS灵魂。
2.  **一人成军的视觉奇观**：当别人还在组队刷怪时，主角身后站着深渊魔龙、堕落天使、亡灵君主……一人即是天灾。
3.  **杀伐果断不圣母**：对于阻碍者和反派，主角信奉“斩草除根”，不仅要在肉体上消灭，还要将灵魂炼制成傀儡，永世为奴。
4.  **BOSS收集癖**：类似宝可梦的收集快感，但对象是恐怖、巨大、压迫感极强的深渊BOSS。

## 主角
-   **姓名**：江寒（Jiang Han）
-   **年龄**：18岁
-   **身份**：江海市第三中学高三贫困生 -> 唯一的“神话级”傀儡君王。
-   **外貌**：黑发黑瞳，身形略显单薄，但眼神极度锐利，常年穿着洗得发白的校服（前期），后期身披暗金长袍，周围环绕黑色以太。
-   **性格特点**：
    -   **极度护短**：平日里冷静低调，但只要有人触碰他的底线（如唯一的妹妹），会瞬间化身修罗。
    -   **利益至上**：不见兔子不撒鹰，所有的行动都有明确的变强目的。
    -   **隐忍后的爆发**：习惯先示敌以弱，收集情报，然后一击必杀。
-   **核心动机/目标**：
    -   **表层目标**：赚钱给身患“魔能侵蚀症”的妹妹治病。
    -   **深层目标**：在万族入侵的末世中活下去，并站上最高的神座，探寻父母失踪的真相。

## 配角（重要配角）

1.  **江小鱼（妹妹）**
    -   **介绍**：主角唯一的软肋，身患绝症但懂事得让人心疼。后期觉醒“圣灵”体质，成为不仅能治疗主角，还能给所有傀儡加持狂暴BUFF的顶级辅助。
    -   **作用**：主角的情感锚点，也是主角疯狂变强的最初动力。

2.  **王胖子（王得发）**
    -   **介绍**：富二代，隐藏职业“多宝商人”。主角的高中死党，负责处理主角打出的海量战利品，并为主角提供情报。
    -   **作用**：搞笑担当，钞能力支援，处理杂务，让主角能专心战斗。

3.  **林清雪（高冷校花/竞争对手）**
    -   **介绍**：觉醒S级职业“冰霜剑姬”，天之骄女。起初对主角的废物职业表示遗憾，后被主角实力折服。
    -   **作用**：侧面衬托主角的强大（爽文常见工具人），后期成为主角军团侧翼的冲锋队长。

## 世界观规则
-   **核心设定**：
    -   百年前，游戏与现实融合，蓝星面积扩大万倍，魔物肆虐，秘境降临。
    -   人类在18岁时进行“转职觉醒”，获得数据化面板。
    -   职业分级：F（生活类）到 SSS（神话类）。
-   **关键规则**：
    -   **傀儡师的诅咒**：普通傀儡师需要耗费天价材料制作傀儡，且傀儡一旦损毁极难修复。
    -   **主角的外挂（金手指）**：天赋【神之手（唯一）】。
        -   *效果1*：**百分百提取**。击杀任何生物，可强制提取其灵魂与尸体，1秒炼成傀儡。
        -   *效果2*：**无限忠诚**。傀儡保留生前战斗本能与技能，但绝对服从。
        -   *效果3*：**属性反哺**。傀儡属性的10%直接叠加在主角身上（后期主角本体一拳也能打爆巨龙）。

## 主线
-   **主线目标**：从底层爬升，组建一只能够横推诸天万界的“神魔傀儡军团”，最终斩杀入侵蓝星的“外神”。
-   **分阶段目标**：
    1.  **觉醒篇**：转职被嘲讽 -> 独自进入新手副本 -> 炼化第一只BOSS（嗜血狼王） -> 高考一鸣惊人，夺得省状元。
    2.  **学院篇**：进入顶级学府 -> 遭遇世家子弟打压 -> 反杀并炼化大家族长老 -> 探索S级禁区，炼化人型BOSS（堕落剑圣）。
    3.  **国战篇**：万族战场开启 -> 外国职业者围剿龙国 -> 主角一人守一城，释放百万傀儡，屠灭敌国精锐。
    4.  **神战篇**：深渊魔神降临 -> 全球绝望 -> 主角将死去的魔神炼化，反攻深渊，登临神座。

## 禁写规则
1.  **禁止虐主**：主角可以陷入苦战，但不能窝囊，不能被小人长久欺压而不还手。
2.  **禁止圣母**：对于试图杀害主角的敌人，禁止出现“原谅”、“感化”情节，必须斩草除根。
3.  **禁止送女**：重要女性角色不能被反派玷污或送给他人，也不能出现令人不适的NTR暗示。
4.  **禁止战力崩坏**：主角可以越级挑战，但不能无逻辑地秒杀高出几个大境界的神明（需要铺垫或利用特殊机制）。
5.  **结尾禁止**：禁止“这只是一场梦”或“主角失去力量归隐山林”的结局。

## 爽点节奏
-   **整体基调**：压抑（10%） -> 爆发（60%） -> 收获/震惊（30%）。
-   **节奏安排**：
    -   **黄金三章**：第一章觉醒废物职业受尽冷眼；第二章激活系统进入副本；第三章首杀BOSS，获得强力打手，瞬间逆转战力。
    -   **小高潮（每10-15章）**：无论是在学校、野外还是拍卖行，安排一个看不起主角的反派（富二代/公会精英）进行挑衅，主角迅速打脸。
    -   **大高潮（每50章）**：大型副本或兽潮攻城。所有人都束手无策，主角登场，召唤新获得的超巨型BOSS，力挽狂澜，享受全人类的膜拜。
    -   **期待感设置**：每次击杀BOSS后，花费篇幅描写新傀儡的强大属性和技能，让读者期待下一次战斗中新傀儡的表现。
